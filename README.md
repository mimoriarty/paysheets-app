# Paysheets App

A React frontend application for a chat interface with an AI agent. Users can send text messages and share files like PDFs with the AI.

## Features
- Chat UI for text messaging
- File upload support (PDFs and other files)
- AI agent integration (to be implemented)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:5173`

## Project Structure
- `src/App.jsx`: Main application component
- `src/App.css`: Styles for the app
- `src/main.jsx`: Entry point

## Development
This project uses agent-driven development with Copilot instructions in `.github/copilot-instructions.md`.
